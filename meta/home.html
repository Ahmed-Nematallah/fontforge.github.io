FontForge is a font editor.

Create and edit fonts in many formats: OpenType, TrueType, AAT, PostScript, Multiple Master, CID-Keyed, SVG and various bitmap formats.

FontForge is libre software: you can share copies freely and make changes under a BSD-style license. You are invited to participate in this project; your bug reports, feature suggestions, tutorial ideas and code contributions are all welcome! Learn more...

Welcome

Learning FontForge is easy. New users can get started with the tutorials, and the entire interface is fully documented.

Font conversion is simple with FontForge. See this tutorial on how to convert font formats.

Scripting is an essential part of any font editor, and FontForge has excellent scripting support, with both its own scripting language and Python.

You can customize FontForge simply too, from hotkeys to the visual theme and translation of the interface into your language.
